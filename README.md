# DocAssistant

AI-powered platform for building embeddable chatbots from company documentation.

## Overview

DocAssistant is a landing page and clickable web application prototype that demonstrates the process of creating an AI chatbot from company documents.

The prototype includes:

- Landing page
- Chatbot Builder
- Knowledge Base
- Train & Review
- Test Chat
- Widget Builder
- Publish page
- Pricing / Paywall
- Responsive layout
- Micro-interactions and animations
